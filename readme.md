# Rails Go to Spec extension for VSCODE

Jump between code and spec in Rails projects.

## Default keybinding:

- Ctrl + Shift + y
- Cmd + Shift + y (Mac)

## Redine shortcuts:


In keybindings.json

```
  ...
	{
		"key": "shift-cmd-y",
		"command": "extension.railsGoToSpec",
		"when": "editorFocus"
	}
	...
```
